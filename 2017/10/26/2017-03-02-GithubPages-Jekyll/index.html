Github Pages + Jekyll 搭建个人博客

发表于 2017-10-26

前序

写 blog 的三种境界

  • 第一阶段，刚接触 blog，觉得很新鲜，试着选择一个免费空间来写；
  • 第二阶段，发现免费空间限制太多，就自己购买域名和空间，搭建独立博客；
  • 第三阶段，觉得独立博客的管理太麻烦，最好在保留控制权的前提下，让别人来管，自己只负责写文章。

今天所要讲的“ Github Pages + Jekyll 搭建个人博客”方法就是让你尽快过渡到第三阶段的利器。

Github & Github Pages

Github, 简单说，它是一个具有版本管理功能的代码仓库，每个项目都有一个主页，列出项目的源文件。
Github Pages，可以被认为是用户编写的、托管在github上的静态网页。
